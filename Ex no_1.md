## Algorithm
1. Start 
2. Declare a variable value of type int 
3. Initialize value with value 5.8. 
4. Print the value of value using printf. 
5. End .   

## Program:
```
#include<stdio.h> 
int main() 
{ 
float a =5.800000; 
printf ("%.1f",a); 
return 0; 
 
}
```
